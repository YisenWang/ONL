# ONL
Code for CVPR2018 "Iterative Learning with Open-set Noisy Labels" https://arxiv.org/abs/1804.00092

## Requirements
- Python 3.8.8
- Tensorflow 2.4.1 
- Keras 2.4.3

## Usage
Simply run the code by python3 train_model_openset.py
